## 1. pwd

### Description
The `pwd` command *shows* the current directory

### Command
```bash
pwd

```
![typing a command in the terminal](pwd.png)

## 2. ls

### Description 
The `ls` command *displays* a list of folders and files

### Command 
```bash
ls

```
![typing a command in the terminal](ls.png)

## 3. ls -la

### Description
The `ls -la` command *displays* deteiled list of files and permissions

### Command 
```bash
ls -la 

```
![typing a command in the terminal](ls-la.png)

## 4. cd /dir

### Description 
The `cd /dir` command is needed to *navigate* to the desire directory

### Command
```bash
cd /

```
![typing command in the terminal](cd0.1.png)

## 5. cd ..

### Description 
The `cd ..` command is need to go to *higher level directory*

### Command
```bash
cd ..

```
![typing command in the terminal](cd0.2.png)

## 6. cd ~

### Description 
The `cd ~` command goes to the *home directory* 

### Command
```bash
cd ~

```
![typing command in the terminal](cd0.3.png)
